Simple Migrator Mode Diagnostic

How to Use

Open the Simple Migrator admin page in WordPress, then open the browser console (F12) and paste the following code:

// === Simple Migrator Diagnostic ===
console.log('=== Simple Migrator Diagnostic ===');

// Check if smData is defined
if (typeof smData === 'undefined') {
    console.error('smData is NOT defined - plugin JS not loaded');
} else {
    console.log('smData defined:', JSON.stringify({
        apiUrl: smData.apiUrl,
        ajaxUrl: smData.ajaxUrl,
        homeUrl: smData.homeUrl
    }, null, 2));
}

// Check UI object
console.log('UI object:', typeof UI !== 'undefined' ? 'loaded' : 'NOT loaded');

// Check mode
var modeInput = document.getElementById('sm-current-mode');
console.log('Current mode:', modeInput ? modeInput.value : 'NOT FOUND');

// Check panels
['sm-source-panel', 'sm-destination-panel', 'sm-no-mode-panel'].forEach(function(id) {
    var el = document.getElementById(id);
    console.log(id + ':', el ? (el.style.display === 'none' ? 'hidden' : 'visible') : 'NOT FOUND');
});

console.log('=== Diagnostic Complete ===');

Manual Testing Commands

Try these commands in the browser console on the admin page:

  • jQuery('.sm-mode-buttons button').first().click() - Click first mode button
  • UI.setMode('source') - Set source mode
  • UI.showCurrentMode() - Refresh mode display
  • jQuery('#sm-source-panel').show() - Force show source panel

Note: This file is for diagnostic reference only. Do not load it directly in a browser.
